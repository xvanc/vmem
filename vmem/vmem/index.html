Vmem Resource Allocator

Vmem is a general-purpose resource allocator which provides constant-time performance with low fragmentation.

For a proper explanation of the allocator, see the paper Magazines and Vmem: Extending the Slab Allocator to Many CPUs and Arbitrary Resources.

Allocation Strategies

Each allocation request can specify an allocation strategy.

See the documentation for AllocStrategy for information on the individual allocation strategies available.
